# GreenPrint-Carbon-Tracker
GUI-based Java application to track and offset carbon emissions

**Overview**  
GreenPrint Carbon Tracker is a JavaFX-based application that helps users track, calculate, and offset their carbon footprint across transportation, food, and lifestyle activities. By providing interactive dashboards, persistent state management, and actionable insights, GreenPrint encourages environmentally conscious decisions and raises awareness about personal carbon emissions.

---

## Features

- **Live Dashboard:** Visualizes carbon footprint data in real-time.  
- **Input & Operations Tab:** Add daily activities and calculate their carbon impact.  
- **Carbon Offset Tab:** Provides recommendations to offset emissions and reduce environmental impact.  
- **Persistent State Management:** Automatically saves and loads user data across sessions.  
- **Logging:** Tracks user operations for transparency and analysis.  
- **User-friendly GUI:** Built with JavaFX for a smooth and interactive experience.

---

## Technologies Used

- Java 17+  
- JavaFX (GUI)  
- File-based persistence (`greenprint_state.txt`, `greenprint_log.txt`)  
- Modular code structure for easy maintenance and future updates

---

## Installation & Running Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/GreenPrint-Carbon-Tracker.git
   cd GreenPrint-Carbon-Tracker/src
Make sure you have Java 17+ and JavaFX installed.

Compile all Java files:

javac *.java

Run the main GUI application:

java GreenPrintGUI

Note: For Visual Studio Code users, configure your JavaFX library in the project settings, then run GreenPrintGUI.java directly.
